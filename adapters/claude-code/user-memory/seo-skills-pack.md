# SEO Skills Pack

Global SEO skill layer installed for Claude Code.

Available user-level skills:

- `/seo`
- `/seo-coral`
- `/ahrefs`
- `/programmatic-seo`
- `/dejan-ai-reverse-engineering`

Default usage:

- Use `/seo` for general technical SEO, content briefs, and search strategy.
- Use `/dejan-ai-reverse-engineering` for AI search, grounding, citation analysis, Google AI Mode, and DEJAN-style reverse engineering.
- Use `/ahrefs` when the workflow is export-driven, backlink-heavy, or competitor-gap oriented.
- Use `/programmatic-seo` when designing scalable page systems and feasibility checks.

If `seo-vector-snapshot` is also installed, use its vector query skills to search the local SEO memory graph before answering deeper research questions.

Prefer memory-backed recommendations over generic SEO advice when the installed skills contain a relevant note or reference.
