## Core Concept
Hobo’s post-leak link building model argues that link value is dynamic, not static, and is continuously requalified by source authority, user validation, and time.

## Key Takeaways
- `siteAuthority` is treated as a composite quality score, not a simple roll-up of backlinks.
- Homepage PageRank still matters because every new page appears to inherit a baseline level of trust from the homepage.
- Hobo interprets modern PageRank through `PageRank-NearestSeeds`, meaning the most valuable links are those closest to trusted seed sites.
- User clicks act as a gate on link value. A link on a page with no real user engagement may pass little or no ranking value.
- Link building therefore becomes less about raw acquisition volume and more about trust proximity, source quality, and real audience validation.
- Cleaning weak pages and weak sections can improve the site-level authority context that determines how links are ultimately valued.

## Frameworks/Methods Covered
- **Authority trinity**: sitewide quality + homepage trust + proximity to trusted seeds
- **User-validation gate**: clicks determine whether a linking page’s outbound links matter
- **Trust map model**: link acquisition should close distance to known authoritative sources

## Data/Stats
- Published: October 14, 2025
- Article references over 2,500 pages and 14,014 leaked attributes as context for the theory
- Hobo specifically connects `siteAuthority`, homepage PageRank, `PageRank-NearestSeeds`, `linkIncoming`, `linkOutgoing`, and `TotalClicks`

## How The Team Should Use This
- **Coral (SEO)**: Prioritize high-trust homepage and category links that lift the whole domain, not only isolated deep-page links.
- **Plankton (Writer)**: Create assets worthy of links from trusted niche authorities, not generic roundup bait.
- **Developer / Chitin**: Strengthen internal distribution from homepage and hub pages so newly published content inherits maximum baseline trust.
- **Current (Marketing)**: Campaigns should target real publishers with real audiences, because user-validated links appear more durable.
- **Key insight for the team**: The right link is valuable because trusted people use it, not just because it exists in HTML.
